  • Links multiple Docker engines

Docker Services

  • Desired state engine which orchestrates containers
  • Service discovery

Docker Compose

  • Orchestrates creation of services
  • Multiple services composed together are a stack
  • Defined in a YML file
  • Manages volumes and networking
